<li> Funções:
  
  `drawBox()`
  
  > Cria um quadrado com base na cor entre parênteses. Por exemplo, drawBox(red) desenha um quadrado vermelho.

CÓDIGO DE EXEMPLO:
  
 
```js
drawBox(blue); //Esse código desenhará um quadrado azul.
```
